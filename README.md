# Space Systems Modelling Portfolio

## Overview
This repository contains selected projects focused on modelling physical systems, analysing signals, and interpreting data under uncertainty. The work combines physics, numerical simulation, and data analysis to explore how complex systems behave under varying assumptions.

## Projects

### 1. Directional Dark Matter Detection (Primary Piece of Work)
- Helped develop a Python-based simulation of recoil energy spectra and directional event rates  
- Analysed sensitivity to laboratory velocity configurations ([0, 220, 0] and [30, 220, 20])  
- Generated sky maps illustrating anisotropic signal structure  
See: `Dark-Matter-Models/`


### 2. Interferometric Image Reconstruction
- Analysed spatial frequency content using 2D Fourier transforms  
- Simulated uv-plane sampling for a 128-element interferometer  
- Reconstructed images from partial frequency data  
See: `Interferometry/`


### 3. Engineering Portfolio
- Morphing wing design and wind tunnel testing  
- Aircraft and rotorcraft assembly  

See: `Engineering-Portfolio/`
